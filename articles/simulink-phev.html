插电式混合动力城市客车动力系统匹配与控制策略仿真【Simulink完整模型】

2026-05-20 · 👁 5678 · 👍 91  ·  Simulink新能源车控制策略能量管理

本文介绍插电式混合动力（PHEV）城市客车的动力系统匹配与控制策略仿真，基于 Simulink 搭建完整模型，适合车辆工程、新能源、自动化方向毕业设计。

一、动力系统结构

以发动机 + 电机 + 动力电池的并联 / 混联结构为例，先做整车需求功率分析，再匹配发动机、电机和电池的参数。

二、参数匹配

  • 根据最高车速、加速性能、爬坡度计算需求功率
  • 匹配电机额定 / 峰值功率与发动机功率
  • 按续驶里程和功率需求确定电池容量

三、能量管理控制策略

实现基于规则的能量管理策略（含 CD/CS 阶段切换），并预留基于优化的策略接口，便于对比。核心逻辑示意：

% 简化的规则型能量管理（示意）
if SOC > SOC_low && P_req < P_eng_opt
    mode = "EV";          % 纯电
elseif P_req > P_eng_max
    mode = "Hybrid";      % 混合驱动
else
    mode = "Engine";      % 发动机为主
end

四、仿真结果

在标准工况下输出车速跟随、SOC 变化、油耗与能耗曲线，用于评估策略效果。模型参数（电机功率、电池容量、减速比等）均可按你的题目修改，附运行说明文档。
